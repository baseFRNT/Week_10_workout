# P2S2: 투자 성향 기반 모의 투자 시뮬레이터

> **Personalized Portfolio Simulation System (P2S2)**  
> 사용자의 투자 성향을 기반으로 개인 맞춤형 포트폴리오를 추천하고,  
> 실시간 모의 투자 및 성향 분석을 제공하는 Flask 웹 애플리케이션입니다.

---

## 프로젝트 개요
| 항목 | 내용 |
|------|------|
| **프로젝트명** | P2S2 (Personalized Portfolio Simulation System) |
| **개발 목적** | 개인의 투자 성향 분석을 기반으로 안전하고 효율적인 투자 학습 환경 제공 |
| **핵심 가치** | 금융 문해력 향상, 투자 습관 개선, 실전 대비형 시뮬레이터 |
| **배포 URL** | http://:5000 |
| **API 엔드포인트** | `/api/<페이지명>` 형태로 JSON 데이터 제공 |

---

## 페이지 구성
| 페이지 | 경로 | 설명 |
|---------|------|------|
| **메인 페이지** | `/main` | 프로젝트 소개 및 링크 |
| **작품 주제** | `/subject` | 프로젝트 주제 및 목표 |
| **실용적 근거** | `/rationale` | 문제 인식 및 필요성 |
| **핵심 기능** | `/features` | 주요 기능 3~5가지 |
| **구현 환경** | `/environment` | 개발 및 배포 환경 정보 |
| **팀 구성** | `/team` | 팀원 역할 및 협업 방식 |

> 모든 페이지는 `/api/<page>` 형태로 JSON 데이터를 반환함.  
> 예: `/api/features` → 핵심 기능 JSON 데이터 출력

---

## 핵심 기능
- 🧩 **투자 성향 분석** : 사용자의 설문 데이터 기반 리스크 선호도 파악  
- 📊 **자산 배분 추천** : 성향에 따라 포트폴리오 비율 자동 추천  
- 💹 **실시간 모의투자** : KOSPI/KOSDAQ API 기반 가상 투자  
- 🧠 **성과 분석** : 투자 결과 시각화 및 리스크 평가  
- 📈 **리포트 제공** : 장기 데이터 기반 투자 습관 분석 리포트 생성
